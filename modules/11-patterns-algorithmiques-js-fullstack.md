# Module 11 — Patterns algorithmiques du développeur JS fullstack

> **Objectif** : transformer les notions précédentes en boîte à outils quotidienne.

> **Difficulté** : ⭐⭐⭐

---

## 1. Sliding window

Très utile pour sous-chaînes, sous-tableaux, quotas, fenêtres temporelles, analytics.

## 2. Two pointers

Excellent quand deux extrémités se rapprochent ou quand une fenêtre doit s'agrandir puis se resserrer.

## 3. Prefix sums

Parfait pour répondre vite à des questions de somme, comptage ou agrégat sur une plage.

## 4. Monotonic structures

Les files ou piles monotones servent pour des maximums / minimums glissants et certains problèmes de monitoring.

## 5. Cas concrets

- rate limiting ;
- anti-spam ;
- classement temps réel ;
- recherche multi-critères ;
- feed avec top K ;
- autosuggest.
