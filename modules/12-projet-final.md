# Module 12 — Projet final

> **Objectif** : assembler les patterns du cours dans un mini toolkit de résolution orienté produit.

> **Difficulté** : ⭐⭐⭐⭐

---

## 1. Le livrable

Construis une petite bibliothèque TypeScript qui expose des utilitaires robustes autour de :

- top K ;
- fusion d'intervalles ;
- BFS sur graphe ;
- memoization ;
- sliding window ;
- union-find.

## 2. Ce qui est évalué

- correction ;
- lisibilité ;
- choix de structure ;
- gestion des cas limites ;
- justification du coût algorithmique.

## 3. Compétence cible

Le but n'est pas de “tout connaître”, mais de savoir **identifier rapidement la bonne famille de solution** puis l'implémenter proprement en TypeScript.
